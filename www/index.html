Arjun Earthperson

Hi, I'm Arjun. I'm currently working on my PhD in Nuclear Engineering at NCSU. My work focuses on Probabilistic Risk Assessment (PRA) under the guidance of Dr. Mihai Diaconeasa.

I was formerly working as a research engineer at the B. John Garrick Institute for the Risk Sciences at UCLA. During that time, I had served as the software development lead for implementing a suite of tools in the probabilistic risk assessment and reliability fields for nuclear, aerospace and power industries. Some of my previous projects include HCLA, a system risk and reliability platform, which is used to enhance the design process and assess the reliability of commercial off-the-shelf (COTS) parts in space systems for extended deep space missions at the Jet Propulsion Laboratory (JPL), and Phoenix, a human reliability analysis tool currently under development for the Japanese Nuclear Regulation Authority (JNRA). In the past, I have worked on numerous machine learning projects in the healthcare, bioinformatics and embedded systems domains.

Projects

    COVID-19 Decision Support Tool

    Simulation tool for exploring COVID-19 scenarios and outcomes for a given community as a function of mitigation measures and policies. Utilizes a hybrid model covering dynamic interactions of epidemiological, psychological, demographic, and policy factors.
  • dehaze

    HCLA

    HCLA is a collaborative, web-based PRA tool based on the Hybrid Causal Logic (HCL) methodology with advanced time-to-failure models, importance measures, and uncertainty quantification. The HCL methodology employs a model-based approach to system analysis. The framework contains a multi-layer structure that integrates event sequence diagrams (ESDs), fault trees (FTs), and bayesian networks (BNs).
  • dehaze

    Phoenix

    Phoenix is a collaborative, web-based human reliability assessment tool. Built on a cognitive human response model, Phoenix incorporates strong elements of current HRA good practices, leverages lessons learned from empirical studies, and takes advantage of the best features of existing and emerging HRA methods.
  • dehaze

    Deriche-Edges-Hough-Transform

    Convex polygon detection using linear hough transform. Polygon detection framework for bitmap images written in C that implements Otsu's thresholding and a Deriche edge detector for blurring, calculation of magnitude and gradient direction, non-maximum suppression and hysteresis thresholding. As a final step, it performs a Hough transform to detect the number and orientation of each polygon edge in the source image.

    Indoor-WiFi-Localizer

    Indoor localization on an Android device using WiFi 2.4/5GHz, Bluetooth and GSM/UMTS/LTE through supervised learning. Uses a kernelized Naive-Bayes classifier implemented using the Weka machine-learning framework.
  • dehaze

    Analog-Utility-Meter-Reader

  • dehaze

    Vowel-Recognition-Engine

    Vowel recognition engine, written in C using PortAudio (compatible with OS X, Windows, Linux). Recognition performed by performing k-means clustering on MFCC features in time-segmented integer audio samples.
  • dehaze

    Smartwatch Activity Recognition

    Real-time detection using supervised learning on Android Wear. Distinguishes between walking, running, lying down, sitting, standing or inactive. 256 extracted features include energy & entropy in time & frequency domains. Classification using deep neural networks performs at ≥85% accuracy in real-world scenarios.
